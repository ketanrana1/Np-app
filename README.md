# Project Name

## Installation

For Backend -:
1. Navigate to backend
2. Install all the packages with command "npm i"
3. Run the server with command "nodemon"

For Frontend -:
1. Navigate to frontend
2. Install all the packages with command "npm i"
3. Run the command "npm start"
